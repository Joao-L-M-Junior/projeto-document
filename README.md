# 🌐 Projeto Documento Base (HTML & JavaScript)

Este é um projeto simples e prático desenvolvido para consolidar conceitos fundamentais de desenvolvimento web, focando na estruturação de documentos com **HTML5**, estilização externa e manipulação básica de eventos do DOM com **JavaScript**.

---

## 🚀 Funcionalidades

*   **Estrutura Semântica:** Uso correto de tags HTML para cabeçalhos e parágrafos.
*   **Vínculo de Estilos:** Integração pronta para arquivos CSS externos (`style1.css`).
*   **Interatividade com JS:** Botão dinâmico que utiliza o método `window.open()` para direcionar o usuário para um link externo em uma nova aba, garantindo uma boa experiência de navegação.

## 🛠️ Tecnologias Utilizadas

*   **HTML5** - Estruturação do conteúdo.
*   **JavaScript (ES6+)** - Lógica de manipulação de eventos.
